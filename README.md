# ⭐ GitHub Stars Tech Radar

**"내가 누른 스타, 다시 지식으로."**
GitHub에서 Star만 눌러놓고 잊어버리셨나요? 이 도구는 쌓여있는 레포지토리를 분석하여 **"지금 바로 쓸 수 있는 지식"**으로 바꿔줍니다.

![Screenshot](https://via.placeholder.com/800x400?text=App+Screenshot)

## ✨ 핵심 기능

- **🔍 AI 인사이트 (DeepSeek/Gemini)**: 레포지토리를 분석해 **"핵심 가치"**와 **"3단계 빠른 시작"** 가이드를 만들어줍니다.
- **🏷️ 목적별 분류**: 단순히 "좋아요"가 아니라 `참고용`, `아이디어`, `실제 사용중` 등으로 목적을 정의합니다.
- **📱 모바일 최적화**: PC, 태블릿, 모바일 어디서든 쾌적하게 볼 수 있습니다.
- **🔒 개인정보 보호**: 별도의 DB 없이 **모든 데이터는 브라우저(=사용자 컴퓨터)에만 저장**됩니다.

---

## 🚀 사용 방법

### 💻 데스크탑 (PC/Mac)
1. `github-stars-guide.html` 파일을 다운로드합니다.
2. 파일을 **더블 클릭**해서 브라우저(Chrome, Edge, Safari)로 엽니다.
3. GitHub 사용자명을 입력하고 **"분석하기"**를 누릅니다.

### 📱 모바일 (아이폰/갤럭시)
모바일 브라우저는 보안상 로컬 파일(`file://`)을 직접 여는 것을 제한하는 경우가 많습니다.  
가장 편한 방법은 **GitHub Pages**를 이용해 **무료로 웹에 올리는 것**입니다.

#### **초간단 배포 방법 (3분 컷)**
1. GitHub에 **새로운 Public Repository**를 만듭니다 (예: `my-stars-radar`).
2. 이 `github-stars-guide.html` 파일의 이름을 **`index.html`**로 바꿔서 업로드합니다.
3. 레포지토리 **Settings > Pages** 메뉴로 이동합니다.
4. **Branch**를 `main` (또는 `master`)으로 설정하고 Save를 누릅니다.
5. 1~2분 뒤 생성된 주소 (예: `https://아이디.github.io/my-stars-radar`)로 접속하면 **내 폰에서도 바로 사용 가능**합니다! 🎉

## ⚙️ 설정 가이드

### AI 분석 기능 켜기 (선택 사항)
AI 기능을 쓰려면 API 키가 필요합니다. 설정 메뉴에서 입력해주세요.
- **Google Gemini**: [무료 키 발급받기](https://aistudio.google.com/app/apikey)
- **DeepSeek V3**: [키 발급받기](https://platform.deepseek.com/api_keys)

> **안심하세요!** 입력한 키는 서버로 전송되지 않고, 오직 내 브라우저에만 저장됩니다.

## 💾 데이터 백업
컴퓨터를 옮기거나 브라우저 캐시를 지울 때를 대비하세요.
- **설정 > 데이터 백업 및 복구** 메뉴에서 `백업 다운로드`를 누르면 모든 데이터가 JSON 파일로 저장됩니다.
- 언제든 `복구` 버튼으로 데이터를 다시 불러올 수 있습니다.
